Getting Started

1. Create the default project {

   npx create-react-app

}

2. Install Semantic UI {

   npm install semantic-ui-react semantic-ui-css

}

3. Install Lodash {

   npm i -g npm
   npm i --save lodash

}

4. Start the App {

   npm start

}
